# Glossaire Informatique - Team_JS

Ce document sert de dictionnaire de référence pour les termes techniques utilisés dans ce projet.

## Développement Frontend
- **React** : Une bibliothèque JavaScript développée par Meta pour créer des interfaces utilisateur basées sur des composants.
- **Vue.js** : Un framework JavaScript progressif pour construire des interfaces utilisateur.
- **Vite** : Un outil de build ultra-rapide qui utilise les modules ES natifs du navigateur.
- **Component (Composant)** : Un bloc de construction d'interface réutilisable (ex: un bouton, une barre de navigation).
- **Virtual DOM** : Une copie légère du DOM réel utilisée par React et Vue pour optimiser les performances d'affichage.
- **Props** : Propriétés passées d'un composant parent à un composant enfant.
- **State (État)** : Données internes d'un composant qui peuvent changer et déclencher une mise à jour de l'affichage.

## Développement Backend & API
- **Node.js** : Environnement d'exécution JavaScript côté serveur.
- **API REST** : Un style d'architecture pour les services web utilisant les méthodes HTTP (GET, POST, etc.).
- **JSON** : Format de données léger utilisé pour l'échange d'informations entre client et serveur.

## Bases de Données
- **SQL** : Langage de requête structuré pour les bases de données relationnelles.
- **NoSQL** : Systèmes de gestion de base de données qui ne suivent pas le modèle relationnel classique.
